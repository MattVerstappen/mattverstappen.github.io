# Lightning
Type: web | Degree: Solo | Date: 2026 | Status: completed
Stack: JavaScript, HTML Canvas, CSS

An interactive lightning system that generates randomised bolt paths using recursive branching segments. Each strike is rendered character by character using '0' and '1' glyphs on an HTML Canvas, producing a stylised binary aesthetic. Branches split from the main bolt at random angles with decreasing intensity. Includes a screen flash on each strike and responds to mouse click events to fire targeted bolts at the cursor position.
