# Black Hole
Type: web | Degree: Solo | Date: 2026 | Status: completed
Stack: CSS

A stylised black hole rendered purely through CSS animations and keyframe transforms. Concentric rotating rings at varied speeds simulate an accretion disc, while layered opacity and blur effects suggest gravitational lensing around the singularity. The piece explores what can be communicated about astrophysical phenomena using only the browser's native styling engine - no canvas, no WebGL, no JavaScript.
