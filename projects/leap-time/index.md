## About

An Android time tracking app with both local and cloud database saves to manage user information. Features include:

- Time entry creation and management
- Timesheet views
- Pomodoro timer
- Audio explanations of each app function
- Local SQLite storage and Firebase cloud sync

Leap Time demonstrates full-stack Android development skills including Kotlin programming, Firebase integration, UI/UX design within Android Studio, and database architecture.
