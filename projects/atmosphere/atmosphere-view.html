Atmosphere

🗓 2026

Live Demo Pure CSS - no interaction required

An ambient particle atmosphere built entirely using CSS keyframe animations. Multiple floating particle layers drift upward at varied speeds and opacities, creating a sense of spatial depth and gentle movement. The effect uses no JavaScript whatsoever, demonstrating what CSS alone can achieve when animation delays, durations, and transforms are composed thoughtfully. Lightweight and performant even on mobile.
