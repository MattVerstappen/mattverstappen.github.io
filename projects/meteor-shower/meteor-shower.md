# Meteor Shower
Type: web | Degree: Solo | Date: 2026 | Status: completed
Stack: JavaScript, HTML Canvas, CSS

A canvas-based meteor shower animation with individually randomised meteors, each assigned a unique trajectory angle, speed, length, and opacity. Meteors leave fading glow trails as they travel and despawn on exit, with fresh ones spawning to maintain consistent density. The system scales particle count to viewport size to stay performant across screen sizes. Built in vanilla JavaScript with no dependencies.
